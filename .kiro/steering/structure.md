# Project Structure

```
.
├── amplify/                  # Amplify Gen2 backend definitions
│   ├── auth/
│   │   └── resource.ts       # Auth config (Cognito + Google provider)
│   └── backend.ts            # Backend entry point
├── src/
│   ├── auth/
│   │   └── AuthProvider.tsx   # Auth context (user state, signIn, signOut)
│   ├── routes/
│   │   ├── __root.tsx         # Root layout (AppBar, nav, loading state)
│   │   ├── index.tsx          # Home page (login prompt or welcome)
│   │   └── profile.tsx        # Protected profile page
│   ├── theme.ts               # MUI theme configuration
│   ├── main.tsx               # App entry (Amplify config, providers, router)
│   └── routeTree.gen.ts       # Auto-generated by TanStack Router plugin
├── amplify_outputs.json       # Auto-generated by Amplify sandbox/deploy
├── vite.config.ts
├── tsconfig.json
└── package.json
```

## Key Patterns

- New pages go in `src/routes/` — the TanStack Router Vite plugin auto-generates the route tree
- Backend resources go in `amplify/` — each resource type gets its own subfolder
- Shared UI components should go in `src/components/`
- Shared hooks should go in `src/hooks/`
