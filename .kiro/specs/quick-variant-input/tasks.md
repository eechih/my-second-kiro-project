# 實作計畫：快速規格輸入（Quick Variant Input）

## 概述

實作「快速規格輸入」功能，包含純函式解析器（`shared/logic/variant-shorthand.ts`）、UI 元件（`src/components/QuickVariantInput.tsx`）、以及與商品新增/編輯頁面的整合。解析器為核心模組，負責將簡寫語法字串解析為 `SpecDimension[]`，並提供格式化輸出與組合數計算。UI 元件提供即時預覽與套用功能。
